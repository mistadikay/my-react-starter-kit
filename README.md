## Prerequisites

- `node@6+`
- `npm@3+`
- (optional) `yarn@0.16.0+`

## Usage

Install dependencies

```sh
npm install
# or
yarn
```

Run development server

```sh
npm start
```

Build for production

```sh
npm run build
```

Run linters (eslint + stylelint + flow)

```sh
npm run lint
```

Run unit-tests

```sh
npm test
```
